Shows posts from Categories as an accordion list with click-able titles. 

== Description ==
Richlist Widget plugin will help you to list WordPress categories in widget areas as an Accordion List. There are some cool options like exclude categories, show/hide read more links, limit the number of posts to be displayed in each category etc etc. This is a really simple plugin, and will do the job :)


== Installation ==
1. Search and install \"Richlist\" plugin.
2. Activate the plugin.
3. Drag and drop the "RichList Posts" widget from menu Appearance->Widgets to desired widget location.
4. To customize the style of the widget, just create a file called "rich-style.css" and put it within the "wp-richlist" folder inside your active theme.
